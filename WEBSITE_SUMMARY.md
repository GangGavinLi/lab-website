## üöÄ Quick Start

### Option 1: Deploy to GitHub Pages (Recommended)

1. **Create a GitHub repository** named `lab-website` (or any name you prefer)

2. **Upload all files** to your repository:
   ```bash
   git init
   git add .
   git commit -m "Initial website setup"
   git branch -M main
   git remote add origin https://github.com/YOUR-USERNAME/lab-website.git
   git push -u origin main
   ```

3. **Enable GitHub Pages**:
   - Go to repository Settings ‚Üí Pages
   - Source: Deploy from a branch
   - Branch: `gh-pages`
   - Click Save

4. **Run first-time setup** (in repository, go to Actions ‚Üí first-time-setup ‚Üí Run workflow)

5. **Your site will be live** at: `https://YOUR-USERNAME.github.io/lab-website/`

### Option 2: Test Locally First

1. **Install dependencies**:
   ```bash
   gem install bundler
   bundle install
   ```

2. **Run locally**:
   ```bash
   bundle exec jekyll serve
   ```

3. **View at**: `http://localhost:4000`
